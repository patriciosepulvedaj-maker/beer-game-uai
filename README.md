# Simulador: El Juego de la Cerveza (Bullwhip Effect)

Este proyecto está dedicado a simular el "Beer Distribution Game" para demostrar el Efecto Látigo en las cadenas de suministro.

## Instrucciones de Uso

Para ejecutar el simulador localmente, ejecuta en la consola de este directorio:

```bash
pip install -r requirements.txt
streamlit run app.py
```
